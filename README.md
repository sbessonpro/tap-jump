# Tap Jump

Un petit jeu mobile en HTML5 / Canvas. Tape l'écran pour faire sauter le personnage par-dessus les obstacles.

## Jouer

Ouvre `index.html` dans un navigateur (mobile ou desktop). Aucune dépendance, aucun build.

- **Mobile** : touche l'écran pour sauter
- **Desktop** : `Espace` ou `↑` pour sauter

## Stack

- HTML5 Canvas
- JavaScript vanilla
- CSS3

Aucune lib externe. Le record est sauvegardé dans `localStorage`.
